# Vision

Claude Code subscriptions enforce rolling rate limits — a 5-hour window and a
weekly (7-day) window — but the consumption against those limits is invisible
during normal work. You only discover you are near a limit when Claude Code slows
or stops, mid-task.

This product gives a developer a permanent, glanceable view of how much of each
limit they have consumed, so they can pace their work and never be surprised by
a limit. It runs locally on the Mac, always on, and stays out of the way until
you look at it.
