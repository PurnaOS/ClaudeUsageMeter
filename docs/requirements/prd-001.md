# PRD

A native macOS app that runs continuously and shows, at a glance, how much of the
Claude Code subscription rate limits the user has consumed.

## Scope (v1)

- A **menu bar item** for the always-on glance, plus a **window** with the fuller
  view when clicked.
- Two gauges: **% of the 5-hour limit** and **% of the weekly (7-day) limit**
  consumed.
- **Time-to-reset** for each window.

## Out of scope (v1)

- Threshold alerts / notifications.
- Historical tracking and graphs.
- Any 1-hour gauge — subscription limits are 5-hour and weekly only; there is no
  1-hour subscription window.

## Data source (resolved)

Per-window consumption and reset times come from the authenticated OAuth usage
endpoint `GET https://api.anthropic.com/api/oauth/usage` — see
[FR-0001](fr-0001.md) and [SPIKE-001](../work/spike-001.md).
